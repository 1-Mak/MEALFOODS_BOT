# MAX MealFoods Bot

Чат-бот для мессенджера MAX, построенный на httpx (async).

Работает в режиме **long polling** — не требует публичного сервера.

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Скопировать и заполнить конфиг
cp .env.example .env
# Отредактировать .env — вписать MAX_BOT_TOKEN

# 3. Запуск
python -m app.main
```

## Переменные окружения

| Переменная    | Обязательна | Описание                        |
|---------------|-------------|----------------------------------|
| MAX_BOT_TOKEN | Да          | Токен бота из MAX для партнёров  |

## Логика бота

- `/start` → приветствие с inline-кнопкой (ссылка)
- Любой текст → эхо `"Вы написали: ..."`
- Нажатие callback-кнопки → уведомление `"Принято"`

## Остановка

`Ctrl+C` — бот корректно завершит работу.
